### 課題

[seq.c](coreutils-9.8/src/seq.c) 

1. print_numebrs 関数内のコメント"`/* If the number …`"について解説しなさい。ソースコード内の処理と関係付けて説明すること。

2. seq_fast 関数は、どのようにfastを実現しているか、処理の流れに基づいて解説しなさい。

3. 次の出⼒を⾏うオプション"-x"を追加しなさい。
- 整数を出⼒するときに、16進数で出⼒する。
- 実数値で出⼒するときはそのまま10進数で出⼒する。(つまり、このケースについては何もする必要はない。)

#### コマンド
設定ファイルの実行（初回）
```bash
./configure
```
ファイルのコンパイル
```bash
make
```
⚠間違えても`make install`をしない

コンパイルしたファイルの場所
```bash
type ./seq
```
コンパイルしたファイルの実行
```bash
./seq
```
⚠`seq`だと`/bin`が実行されてしまう
